# Repository

Repository를 구성하고 이를 upload / download하는 기능을 만든다.

## Repository Server
- repository server는 upload / download에만 충실한다.
- 다양한 API 를 제공한다.
- Cache 기능으로 속도 UP

## Repository Client
- download
- autoupdate
- checksum 
- version별 다운 기능
- version별 설치 경로 지정 및 여러 버전 관리

## Repository Lib
- Repository upload / download

## Requirement
- c#
- windows 10
